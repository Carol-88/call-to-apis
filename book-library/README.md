# Book Library

Este proyecto es una biblioteca virtual donde los usuarios pueden buscar libros, ver detalles y agregarlos a listas personalizadas como "Favoritos", "Por leer" y "Leídos". Utiliza la API de Open Library para obtener información sobre libros.

## Tecnologías Utilizadas
- React
- TypeScript
- Tailwind CSS
- Axios para llamadas a la API

## Instalación
1. Clona el repositorio y navega a la carpeta `book-library`.
2. Instala las dependencias:
   ```bash
   npm install
3. Ejecuta el proyecto:
   ```bash	
   npm start

## Funcionalidades
- Búsqueda de libros por título o autor.
- Visualización de detalles del libro (autor, año, descripción).
- Creación de listas personalizadas: Favoritos, Por leer y Leídos.

## API
Este proyecto usa la API de Open Library: https://openlibrary.org/developers/api
